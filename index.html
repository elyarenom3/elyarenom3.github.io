Experience

My most recent experience as a Software Engineer was for a research team focused on developing a programmable oncogene targeting system, a project that beat over 400 teams to win the international iGem competition.

This involvement was part of an intensive year-long undergraduate project, working full-time during the summer in a cross-functional team. Our goal was to design a CRISPR-based system, named Craspase, capable of inducing pyroptotic cell death in cancer cells.

In this role, I became proficient in utilizing software like PyMOL and Matlab, which were crucial for our project's success. I developed a GitHub pipeline to enhance our team's collaboration and ensure smooth progress. My contributions included designing modular parts for our project, demonstrating its potential applicability in treating a wide range of cancer types and RNA-based diseases. This experience significantly honed my skills in technical communication, problem-solving, and interdisciplinary collaboration.

Outside of this amazing experience, I have also involved myself in numerous hackathons, most recently achieving second place in the McWiCS hackathon.

Recent Projects

F1 fantasy

A web-based F1 fantasy analysis application

Automated race and driver statistic extraction and preprocessing with Python and Pandas. The backend, built with Spring Boot, included RESTful APIs for optimal performance and scalability. A user-friendly front-end was designed using HTML, CSS, and JavaScript, featuring interactive data visualization, real-time updates, and custom comparisons. I used strategic deployments for high availability during peak events. This project showcased my technical expertise and commitment to delivering a complete, userfocused software solution.

PolitiAlign

A political alignment tool

This website aims to promote informed voting and political engagement by providing users with matches based on their political preferences and values. By leveraging advanced data scraping techniques and natural language processing, we gather comprehensive data from official sources such as Open Data by the House of Commons and LegisInfo, analyze the legislative contributions of canadian MP’s and then, using information gathered about the user through a survey, we tell them how much they agree with their MP. Frontend is very user friendly, constructed using HTML and CSS and we used rest API and jSON to connect.

Personal AIssistant

A personal assistant bot

A sophisticated personal assistant application with a ReactJS frontend for intuitive task management and a Node.js backend for real-time synchronization, including synchronized video watching via YouTube API and Web Sockets. I also engineered a Node.js and OpenAI API-powered chatbot, deployed serverlessly on AWS EC2, for scheduling and query assistance. The application was praised for its user-friendly design and productivity enhancement in a university setting and I am planning future API integrations and UI refinements based on user feedback.
